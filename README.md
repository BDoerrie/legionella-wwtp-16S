# legionella-wwtp-16S
R scripts for the analysis of Legionella 16S rRNA amplicon sequencing data and statistical analyses supporting the associated publication "Legionella community diversity and dynamics in activated sludge systems of wastewater treatment plants: Are they transient visitors or residents?"

# Overview

The workflow includes:
-
- Filtering of singleton, doubleton, tripleton, and quadrupleton ASVs from the ASV table generated by the DADA2 pipeline.
- Rarefaction of the ASV table.
- Identification of Legionella ASVs using BLAST results using a custom Legionella reference database.
- Alpha-diversity analyses (Observed richness and Shannon diversity)
- Spearman's rank correlation analyses with wastewater treatment plant (WWTP) parameters, alpha-diversities and Legionella qPCR-concentrations.
- Beta-diversity analyses based on Bray–Curtis dissimilarity.
- Creation of Venn diagrams for zeta-diversity analyses.
- Generation of a summary table listing the Legionella species to which species-assigned amplicon sequencing variants (ASVs) were attributed, together with the wastewater treatment plants (WWTPs) in which they were detected and their detection frequency across sampling events.
- Generation of figures and tables.

# Requirements
The analyses were performed in R 4.5.0.

Required R packages are listed at the beginning of the analysis script and include packages such as:
- vegan
- FSA
- dplyr
- ggplot2
- pairwiseAdonis
- openxlsx
- Biostrings
- patchwork
- VennDiagram
- corrplot
- ggsignif

Input data
-
The script expects the required input files to be located in the corresponding folders within the data/ directory.
Large sequencing datasets are not included in this repository. They are available through the corresponding public repository referenced in the manuscript.

Outputs
-
Running the script produces:
•	Rarefied ASV tables
•	Alpha- and beta-diversity statistics
•	Correlation analyses
•	Venn diagrams
•	TIFF figures

# Reproducibility
The code in this repository corresponds to the analyses used in the accompanying publication.
